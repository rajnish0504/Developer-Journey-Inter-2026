# Basic Git Commands

## Check Git Version
```bash
git --version
```

## Initialize Git Repository
```bash
git init
```

## Check Status
```bash
git status
```

## Add Files
```bash
git add .
```

## Commit Changes
```bash
git commit -m "message"
```

## Push Code
```bash
git push origin main
```

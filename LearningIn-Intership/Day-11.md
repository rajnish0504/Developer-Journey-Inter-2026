# Day 11 - React API Integration using Fetch & Axios

## What I Learned Today

* Practiced API integration in React using both Fetch API and Axios with fake APIs.
* Learned how to fetch data from APIs and display dynamic data on the UI using React components.
* Understood how API responses are handled in React applications.
* Explored the use of `async/await` for handling asynchronous API calls in a cleaner and more readable way.
* Learned about loading states and error handling during API requests.
* Understood the difference between Fetch API and Axios for API handling in React.
* Practiced working with normal fake APIs to understand real-time API integration concepts.
* Performed `GET`, `POST`, `PUT`, and `DELETE` operations using Axios to understand different HTTP methods.
* Learned how to handle API requests and responses for different CRUD operations.
* Practiced displaying fetched API data dynamically on the UI.
* Worked with fake eCommerce APIs and rendered product data using dynamic React cards.
* Created reusable React card components for displaying API data.
* Learned how to map API data and render product details dynamically in React applications.
* Improved understanding of component-based UI rendering with API-driven data.
* Explored better handling of asynchronous operations, loading states, and error management during API calls.
